# PATCH NOTES: V0.1.2

A score display bug fix and a few general gameplay/experience improvements.

## Bug fixes
- fixed a bug where a -1 score display would appear when a damaged object slipped back into the water, if a slow/fast press was performed on a non-target finger (this is only a cosmetic fix, as the actual score would not change)

## General improvements
- runs can be aborted with the escape key at any time (takes you back to shore, for example if you set the difficulty too high or easy)
- moved the finger markers closer together in two handed mode (there is still a 1.5x gap between the hands)
- added a favicon of the shell
