# PATCH NOTES: V0.1.4

Adds a 4-finger mode on desktop, and reworks the mobile control layouts to
support 4- and 5-finger play. No changes to game logic, timing, hitboxes,
or recorded data.

## General gameplay (desktop)
- The finger-count selector now offers **3, 4, and 5** (was 3 or 5).
- **4-finger** = the four non-thumb fingers: little/ring/middle/index
  (Q-W-E-R on the left, U-I-O-P on the right). Comfortable default — no
  thumb, no spacebar, hands rest on the home row.
- This is the only desktop change; everything below is mobile.

## General gameplay (mobile)
- Mobile now offers **4-finger and 5-finger** layouts in addition to 
  the existing 3-button version (4 fingers is now the default).
- **4-finger mobile:** four finger buttons filling the bottom region. They
  extend from the bottom of the screen all the way **up to the bottom of
  the wave** (tall touch targets across that shared space).
- **5-finger mobile:** the thumb becomes a **full-width spacebar-style
  button** across the very bottom, with the four finger buttons sitting
  **above** it and extending upward (again up toward the bottom of the
  wave). This is the "thumb underneath" experimental layout.
